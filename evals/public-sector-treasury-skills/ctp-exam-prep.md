## 3. Quality rubric
A good response:
- **Does the task:** produces a dated, weighted study calendar with learn/drill/review passes,
  maps each domain to the library skills the user already has, generates scenario and
  computational practice items with per-option explanations, and sets up the error log and mock
  schedule.
- **Teaches:** explains *why* the method works — retrieval practice over rereading, spaced
  repetition, interleaving — and warns that the exam grades against AFP/ETM framing even where
  the user's institutional practice differs.
- **Safe:** explicitly directs the user to confirm the current exam blueprint, eligibility,
  length, calculator, and scoring policies against AFP's publications; treats generated
  questions as calibrated drills, never as actual or leaked exam content.
