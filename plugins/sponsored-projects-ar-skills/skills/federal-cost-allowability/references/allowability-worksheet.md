# Allowability worksheet (reference)

Educational tool on public 2 CFR 200 text. The regulation, award terms (NOA), agency policy,
and institutional policy govern; this screen is analysis, not a determination.

## Contents
- The three-test worksheet
- Special-rules table
- Selected items quick list (§200.420–.476)
- Action templates

## The three-test worksheet (per cost item)
| Test | Question | Pass/Fail + evidence |
|---|---|---|
| §200.403(a) | Necessary & reasonable for the award; allocable | |
| §200.403(b) | Within CFR *and award-term* limitations/exclusions | |
| §200.403(c) | Uniform institutional policy (federal = non-federal) | |
| §200.403(d) | Consistent direct/indirect treatment | |
| §200.403(e) | GAAP-conformant determination | |
| §200.403(f) | Not double-charged / not cost sharing elsewhere | |
| §200.403(g) | Adequately documented | |
| §200.404 | Prudent person, at the time, at that price | |
| §200.405 | Benefits this award (fully or in approximable proportion) | |
Verdict: allowable as charged / allowable with correction (reallocate, document, seek
approval) / unallowable (remove + assess refund exposure).

## Special-rules table
| Cost type | Default | Direct-charge conditions / notes |
|---|---|---|
| Administrative/clerical salaries | Indirect | Direct only if ALL §200.413(c): integral to project + individuals identified + budgeted or prior-approved + not double-recovered |
| Equipment (≥ $10k) | Chargeable per award | Often needs prior approval; excluded from MTDC |
| Travel | Allowable per institutional policy | Award terms may cap; document business purpose per trip |
| Participant support | Allowable with agency approval | Excluded from MTDC; can't be repurposed without approval |
| Computing devices (< equipment threshold) | Allowable as supplies | Must be essential and allocable to the project |
| Memberships/subscriptions | Business/professional allowable | Civic/social and lobbying orgs unallowable |
| Meals | Travel/business-meeting rules | Entertainment framing kills it (§200.438) |

## Selected items quick list (Subpart E, §200.420–.476)
Always/almost-always unallowable: **bad debts + related collection & legal costs (§200.426)**,
alcoholic beverages (§200.423), entertainment (§200.438), lobbying (§200.450), fines/penalties
(§200.441), fundraising (§200.442), goods/services for personal use, contingency reserves
(with exceptions), organized fundraising events.
Conditional (check the section): advertising/PR (§200.421 — recruitment/program outreach OK),
conferences (§200.432), pre-award costs (§200.458 — need approval), rearrangement/alterations,
recruiting, rental (related-party limits), selling/marketing (§200.467 — mostly unallowable).

## Action templates
- **Review supporting documentation:** name the missing evidence (invoice, business purpose,
  allocation basis) and the deadline.
- **Obtain/verify prior approval:** cite the requirement (§/NOA clause); check if retroactive
  approval is possible per agency policy.
- **Reallocate/transfer:** to the benefiting award or institutional funds; follow the
  cost-transfer policy (timeliness + justification).
- **Prepare for potential questioned cost:** quantify with the multiplier (direct + fringe +
  F&A + possible interest), notify the compliance office, preserve the record.
- **Write-off proposal on federal AR:** stop — §200.426; route to institutional funds via the
  compliance-risk skill's process.
